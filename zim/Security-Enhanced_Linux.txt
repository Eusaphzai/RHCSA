====== Security-Enhanced Linux ======
Created Saturday 14 July 2018

[[star]]

Security-Enhanced Linux, also known as ([[SELinux]]), is a Linux kernel security module.  It allows Linux administrators to create access control security policies that granularly allow or prevent access to system resources based on user privileges and roles.
