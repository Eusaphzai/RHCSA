====== MBR ======
Created Thursday 19 July 2018

[[BIOS-MBR - legacy boot process:legacy boot process - stage 2 - MBR]]

**Summary**
MBR stands for Master Boot Record. The MBR is located in the first physical sector of a bootable storage device, such as a hard drive. The MBR contains computer code needed for the [[Boot Loader]] to load the operating system kernel into memory.
