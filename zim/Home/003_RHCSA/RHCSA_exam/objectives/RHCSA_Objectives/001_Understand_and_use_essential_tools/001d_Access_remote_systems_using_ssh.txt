**RHCSA Objective: **001d Access remote systems using ssh

**Summary**
Ssh stands for secure shell. Ssh is a command-line interface and protocol that is used to access remote Linux/Unix computers from a Linux/Unix machine. Ssh is part of a 3-part suite of secure applications:
1. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:slogin|slogin]]
2. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:ssh|ssh]]
3. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:scp|scp]]
