====== legacy boot process - stage 3 - GRUB ======
Created Wednesday 18 July 2018

[[BIOS-MBR - legacy boot process]]

**Summary**
[[alphabetical list of essential tools and terms:F-J:G:GRUB]] stands for GRand Unified Bootloader. Its job is to load the operating system kernel into memory, and to then tranfer control to the kernel so that the boot process can be completed. Initializing GRUB is the 3rd stage of the overall boot process, but GRUB itself has three internal stages:
* [[alphabetical list of essential tools and terms:F-J:G:GRUB:GRUB Stage 1]]
* [[alphabetical list of essential tools and terms:F-J:G:GRUB:GRUB Stage 1.5]]
* [[alphabetical list of essential tools and terms:F-J:G:GRUB:GRUB Stage 2]]

More information about GRUB can be found here: 
https://linoxide.com/booting/boot-process-of-linux-in-detail/
