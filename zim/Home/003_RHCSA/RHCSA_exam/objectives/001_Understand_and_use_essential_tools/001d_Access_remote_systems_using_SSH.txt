**RHCSA Objective: **001d Access remote systems using ssh

**Summary**
[[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:P:ports:00022 SSH TCP|SSH]] stands for **Secure Shell**. SSH is a command-line interface and protocol that is used to access remote Linux/Unix computers from a Linux/Unix machine. SSH is part of a 3-part suite of secure applications:
1. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:slogin|slogin]]
2. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:P:ports:00022 SSH TCP|SSH]]
3. [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:scp|scp]]

**remote SSH - examples:**
* [[+001d - example 1]] : Transfer a file to a RHEL/CentOS machine using [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:scp|scp,]] Login to the remote machine via [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:P:ports:00022 SSH TCP|ssh]], and manipulate the newly transferred file via the ssh connection.
* [[+001d - example 2]] : Login to a RHEL/CentOS machine via [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:P:ports:00022 SSH TCP|ssh]], transfer a file to it using [[Home:003 RHCSA:alphabetical list of essential tools and terms:P-T:S:scp|scp]], and manipulate the newly transferred file via the ssh connection.
