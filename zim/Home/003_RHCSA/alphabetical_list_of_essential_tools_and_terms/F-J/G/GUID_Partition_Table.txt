====== GUID Partition Table ======
Created Wednesday 18 July 2018

Home:RHCSA:alphabetical list of essential tools and terms:A-E:B:Boot Loader
[[G]]

**Summary**
The GUID Partition Table is a newer disk architecture that introduces better functionality when compared to the older [[K-O:M:MBR]] (Master Boot Record) partitioning method. For more information on the GUID Partition Table, see here: https://support.microsoft.com/en-us/help/302873/frequently-asked-questions-about-the-guid-partitioning-table-disk-arch
