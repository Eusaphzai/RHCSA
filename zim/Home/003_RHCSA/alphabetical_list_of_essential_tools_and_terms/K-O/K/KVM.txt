**Summary**
KVM stands for Home:RHCSA:alphabetical list of essential tools and terms:K-O:K:Kernel-based Virtual Machine. It is an [[O:open source]] / Home:RHCSA:alphabetical list of essential tools and terms:F-J:F:free software program that allows a host computer system to operate virtual machines running on guest operating systems.  For example, KVM would allow one to run Home:RHCSA:alphabetical list of essential tools and terms:A-E:C:CentOS on a virtual machine hosted on a physical computer running the Home:RHCSA:alphabetical list of essential tools and terms:F-J:F:Fedora operating system.

More information on KVM can be found here:
* https://www.linux-kvm.org/page/Main_Page
* https://en.wikipedia.org/wiki/Kernel-based_Virtual_Machine
