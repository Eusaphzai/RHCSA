**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary:**
The **KornShell** is a [[Home:003 RHCSA:alphabetical list of essential tools and terms:A-E:C:command-line interpreter|command-line interpreter]] for Linux/Unix that was developed by David Korn at Bell Labs in the early 1980s.  The KornShell was initially based on the source code for the [[Home:003 RHCSA:alphabetical list of essential tools and terms:A-E:B:Bourne shell|Bourne shell]].
