====== cpuinfo ======
Created Wednesday 16 January 2019

The cpuinfo file contains information on the computer's microprocessor.  It is found in this directory:
[[/proc/cpuinfo]]

You can view the cpuinfo file as follows:
cat [[/proc/cpuinfo]]
