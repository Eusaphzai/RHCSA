**RHCSA Objective: **001a Access a shell prompt and issue commands with correct syntax

**Summary**
Alias is a Bash built-in command. The alias command allows you to run a command or group of commands by entering a replacement string.  For example, the **run_script1** alias could be used to run a Bash shell script.

**alias:**
* alias - function: replace words with another string.
* alias - description: Alias is often used to abbreviate long system commands using a shorter alias string.
