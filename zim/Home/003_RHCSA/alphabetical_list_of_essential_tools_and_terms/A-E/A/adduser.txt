**KYWRD:** adduser
**RHCSA Objective: **006a Create, delete, and modify local user accounts

**Summary**
adduser is used to create a new user account, and to modify user account settings during the creation process.  The adduser and [[U-Z:U:useradd|useradd]] commands are interchangeable.

**adduser:**
* adduser - function: The adduser command is very similar to the [[U-Z:U:useradd|useradd]] command. It is used to create new user accounts, and to set various account parameters during the creation process.
* adduser - description: The adduser command is very similar to the [[U-Z:U:useradd|useradd]] command. It is used to create new user accounts, and to set various account parameters during the creation process.
