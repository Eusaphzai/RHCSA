====== Rufus ======
Created Monday 23 July 2018

Home:RHCSA:alphabetical list of essential tools and terms:F-J:F:formatting USB drives
[[R]]

**Summary**
Rufus is a utility that allows you to format and create bootable USB flash drives. On the Rufus website, Rufus is downloadable as a Windows .exe file, so the Linux sysadmin could use Rufus to create a Linux boot disk on a Windows machine.  Rufus is free software licensed under the GPL.

More information of Rufus can be found here: https://rufus.akeo.ie/
