**KYWRD:** ssh
**RHCSA Objective: **001d Access remote systems using ssh

**Summary**
This article lists the options available with the [[ssh]] command.

**KEYWORD - options:**
* −4: forces **ssh** to use IPv4 addresses exclusively.
* −6: forces **ssh** to use IPv6 addresses exclusively.
* −A: allows forwarding of the authentication agent connection.
* −a: disables forwarding of the authentication agent connection.
* −B //bind_interface: //tells **ssh** to bind to the address of the **bind interface** before connecting to the destination host.
* −b //bind_address: //tells **ssh** to use the **bind address** of the local machine as the source address of the connection.
