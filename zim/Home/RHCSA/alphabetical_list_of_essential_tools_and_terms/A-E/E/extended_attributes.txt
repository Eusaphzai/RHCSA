**KYWRD:** extended attributes

**Summary**
Extended file attributes are attibutes in a Linux filesystem that go beyond the common attributes such as file permissions.  Extended attributes are often called EA of xattrs. 

The extended attributes on Linux have four namespaces:
* user
* trusted
* security - The security namespace is used by SELinux.
* system 

**extended attributes:**
* extended attributes - function: extended attributes (EA) allow system users to associate files with metadata that is not interpreted by the Linux filesystem.
