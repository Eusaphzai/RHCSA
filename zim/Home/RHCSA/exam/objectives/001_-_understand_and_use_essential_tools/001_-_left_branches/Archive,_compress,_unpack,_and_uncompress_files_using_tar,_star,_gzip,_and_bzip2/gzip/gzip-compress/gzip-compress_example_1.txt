====== gzip-compress example 1 ======
Created Sunday 22 July 2018

:Home:RHCSA:exam:objectives:001 - understand and use essential tools:001 - left branches:Archive, compress, unpack, and uncompress files using tar, star, gzip, and bzip2:gzip:gzip-compress example 1:gzip:gzip-compress

**Summary**
gzip can only compress one file at a time.  You can compress multiple files like this:
* gzip filename1 filename2 ...filenamez
* gzip -r directory-name

The second command above will compress all of the files in the directory directory-name.  For example, if directory-name contains myfile.1.odt, myfile2.odt and myfile3.odt, gzip -r directory-name will result in directory name containing myfile.1.odt.gz, myfile.2.odt.gz and myfile.3.odt.gz.  Let us see this in action.

Tasks
* Navigate to [[~/Sync/PROJECTS/RHCSA/examples/gzip/gzip-compress_ex001]]
* Create a new directory called my_libreoffice_files
* [[cd]] to my_libreoffice_files and create myodt.1.odt through myodt.8.odt => touch myodt.{1..8}.odt
* Move up one level, and compress all of the .odt files using this command: gzip -r my_libreoffice_files
* Note that myodt.1.odt.gz through myodt.8.odt.gz are created inside my_libreoffice_files
* Note that the compressed version of the files replaces the original versions.

If you want to include more than one file in a gzip archive, you have to roll the files up into a single tarfile.  In gzip-compress example  2, we will first create a set of .odt documents.  We will then roll them up into a .tar archive. Then we will use gzip to compress that tarfile.
