**KYWRD:** EFI System Partition

**Acronym:** [[+ESP|ESP]]

**Summary**
The EFI System Partition (ESP) is a hard drive partition which stores files that are loaded by the Unified Extensible Firmware Interface (UEFI) during the boot process. The ESP contains boot loaders or kernel images for all installed operating systems on the computer system.  The ESP is identified by the globally unique identifier in the GUID Partition Table (GPT).
