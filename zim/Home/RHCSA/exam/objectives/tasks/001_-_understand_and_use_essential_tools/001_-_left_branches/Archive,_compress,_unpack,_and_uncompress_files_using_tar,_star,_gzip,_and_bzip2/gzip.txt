**Summary**
gzip compresses the named file using the [[Lempel-Ziv coding]] ([[LZ77]]), la ossless compression algorithm  using variable-rate coding created in 1977 by [[+gzip-compress:Abraham Lempel]] and [[Jacob Ziv]]. gzip compression is also employed when the -z flag option is applied to the [[tar]] command. See [[tar-c`ompress - example 1]] for more details.
