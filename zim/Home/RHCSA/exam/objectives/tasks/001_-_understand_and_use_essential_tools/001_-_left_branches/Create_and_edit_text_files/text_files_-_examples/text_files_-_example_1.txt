**Summary**
In text files - example 1, we will create some simple empty text files on our [[alphabetical list of essential tools and terms:A-E:C:CentOS]] VM using the [[files:touch]], [[mkdir]], [[files:ls]] and all templates:options template:alphabetical list of essential tools:C:cd commands.

**text files - tasks **(command line commands are __highlighted__)**:**
* Open the Virtual Machine Manager and startup your RHEL/CentOS virtual machine.
* Log in to your virtual machine as a normal user.
* Create a new "Examples" directory in your home directory on the virtual machine (mkdir with the -p option allows you to create a directory and subdirectories all in a single command.): __mkdir -p Examples/text_files/example_001__
* all templates:options template:alphabetical list of essential tools:C:cd to the new example_001 sub-directory: __cd Examples/text_files/example_001__
* Create 20 new empty text files named status_report001.txt through status_report020.txt using the [[files:touch]] command: __touch status_report{001..020}.txt__
* Use the ls command to verify that the 20 new text files were created: __ls__
