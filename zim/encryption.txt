====== encryption ======
Created Tuesday 10 July 2018

[[004 - create and configure file systems]]

Linux files or folders can be encrypted using the  [[GnuPG]] utility.  [[GnuPG]] is used to encrypt or decrypt files and folders via the [[gpg]] command.  The general syntax for encrypting a file with gpg is as follows:
* gpg -c document_name

Examples
* Please see [[gpg-encrypt example 1]] for more details on encrypting files using GnuPG.
