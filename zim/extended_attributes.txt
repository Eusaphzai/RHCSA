====== extended attributes ======
Created Tuesday 17 July 2018

[[star example 2]]

Extended file attributes are attibutes in a Linux filesystem that go beyong the common attributes: the file permissions.  Extended attributes are often called EA of xattrs. 

The [[extended attributes]] on Linux have four namespaces:
* user
* trusted
* security - The security namespace is used by SELinux.
* system
