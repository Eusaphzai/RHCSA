====== initrd ======
Created Thursday 19 July 2018

[[Home:RHCSA:exam:objectives:002 - operate running systems:002 - right branches:Boot, reboot, and shut down a system normally:boot:BIOS-MBR - legacy boot process:legacy boot process - stage 4 - Kernel]]

**Summary**
initrd stands for [[initial RAM disk]]. it provides an initial root filesystem that is mounted prior to the "real" filesystem being mounted. initrd is initialized in concert with the kernel initialization.

For more information on initrd, visit here:
https://www.ibm.com/developerworks/library/l-initrd/index.html
