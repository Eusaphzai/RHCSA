====== bzip2-description ======
Created Thursday 12 July 2018

[[Home:RHCSA:exam:objectives:001 - understand and use essential tools:001 - left branches:Archive, compress, unpack, and uncompress files using tar, star, gzip, and bzip2:bzip2]]

bzip2- compresses or decompresses single files
* free software/open source file compression program
* uses [[Burrows-Wheeler algorithm]]
* cannot combine multiple files into a single archive
* only used to compress single files
* to compress an archive of mulitple files, you would first have to compress the files in a single archive using a different compression algorithm first.  Then this single archive file could be further compressed using bzip2.
