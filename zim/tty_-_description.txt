====== tty - description ======
Created Wednesday 11 July 2018

[[tty]]

The tty command prints the file name of the terminal connected to the standard input (the keyboard).

It has the following flags:
* -s, --silent, --quiet
	* prints nothing, only returns an exit status
* --help
	* displays help information then exits
* --version
	* displays version information then exits
